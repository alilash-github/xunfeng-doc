# 巡风之任务功能

任务功能是建立在巡风扫描系统已经有资产的基础上，使用搜索功能搜索出目标主机，使用系统已有的插件进行周期性扫描。

步骤：

1. 利用搜索功能搜索出目标主机

   ![](../img/3.3_01.png)
   或进入插件功能

   ![](../img/3.3_02.png)
2. 添加任务

   ![](../img/3.3_03.png)
3. 填写任务各个项信息

   ![](../img/3.3_04.png)

4. 保存
    ![](../img/3.3_05.png)
